# Express Project Template
First node.js + express project template

## How to setting

### 1. Clone project from github.

```console
$ git clone git@github.com:wooyaggo/express.git
$ cd express
```

### 2. Install modules.

```console
$ npm i
```

### 3. Run project.

Using pm2 module.

```console
$ npm start
```

### 4. See logs.

```console
$ pm2 log
```
